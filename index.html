Preventing Zero-Day Vulnerabilities on Software Vendors by Scanning Its Malware Using a Tree Traversal Algorithm

Research by: Delaney Campos - Ramirez

Background

  • Web developers don't realize there are vulnerabilities in their code until they are hacked into
  • With the increase in advanced technology, there are more cases of online hackers
  • Since Python is a versitile language, it is used in cyber-security to stop hackers

Rationale

  • A Tree-Traversal Algorithm contain three different types of algorithms, all of which work differently but they all check the entire code
    1. PreOrder
    2. PostOrder
    3. InOrder
  • This study uses Python and the three types of Tree-Traversal Algorithms to see if they can detect the vulnerabilities in a website accurately.

Conclusion

  • Out of the three Algorithms, the PreOrder Algorithm proved to be the most accurate
  • However, all three of them have the strenths and weaknesses
